## IMOBot
7th Cavalry's IMO Discord Bot

# How to run:

1. fill in config/main.json
2. npm run start

# Features:
  * Connected to 7th Cav API
  * Configurable Tokens
  * Configurable Billets
  * Configurable Ranks

# Chat Commands:
  * !Milpac <DiscordUser>
    Desc: Returns a link to specified user's Milpac.
